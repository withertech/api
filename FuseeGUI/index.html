Fusee GUI

For the Raspberry Pi


Fusée GUI is a Graphical User Interface for the Nintendo Switch
vulnerability known as “Fusée Gelee”. The GUI allows for easy injection
of payloads. Included is the hello world payload and more can be imported
with the import dialog.

To start the GUI, simply type this into a terminal:
pi@raspberrypi ~ $ Fusee-qt
